Guia de Integração com a Shopee

📋 Passo a Passo — Como integrar com a Shopee

  1. Crie uma conta de desenvolvedor Shopee: Acesse open.shopee.com e faça login com sua conta Shopee. Clique em "Apply to be a developer".
  2. Crie um App: No painel, clique em "Create App". Dê um nome (ex: Print3D Dashboard), selecione o tipo "Self-developed" e preencha a URL do seu sistema.
  3. Obtenha as credenciais: Após aprovação, você receberá o Partner ID e o Partner Key. Guarde com segurança.
  4. Autorize sua loja: Gere o link de autorização abaixo e acesse-o para vincular sua loja Shopee ao sistema. Isso gera o Shop ID e o Access Token.
  5. Configure o Webhook: No painel do app Shopee, vá em "Configure" → "Webhook" e adicione sua URL de recebimento. O sistema enviará notificações para cada novo pedido.
  6. Teste a integração: Faça um pedido de teste na sua loja ou use o Sandbox da Shopee para verificar se os pedidos aparecem aqui no sistema.
  7. Sincronização automática: Após configurado, novos pedidos da Shopee aparecem automaticamente na Central de Pedidos com a origem marcada como SHOPEE.

🔑 Credenciais necessárias

📡 URL do Webhook (Copie e cole no painel da Shopee)

https://seudominio.com/webhook/shopee

Esta URL receberá os eventos da Shopee (novos pedidos, cancelamentos, atualizações de status).

⚙️ Exemplo de código — Receber Webhook Node.js

const express = require('express'); const crypto = require('crypto'); const app = express(); app.use(express.json()); app.post('/webhook/shopee', (req, res) => { // Verificar assinatura const signature = req.headers['authorization']; const body = JSON.stringify(req.body); const expected = crypto .createHmac('sha256', process.env.SHOPEE_PARTNER_KEY) .update(body) .digest('hex'); if (signature !== expected) { return res.status(401).json({ error: 'Unauthorized' }); } const { code, data } = req.body; // Novo pedido if (code === 3) { const orderId = data.ordersn; console.log('Novo pedido Shopee:', orderId); // Salvar no seu banco de dados // notificarSistema(orderId); } res.json({ status: 'ok' }); }); app.listen(3000);

        "Access-Control-Allow-Origin": "*",
        "Access-Control-Allow-Methods": "GET"
